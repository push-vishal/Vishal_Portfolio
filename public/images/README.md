# Your portfolio images

Copy your photos from your PC into **this folder** using these exact filenames:

| Filename | Used in | Suggested aspect |
|----------|---------|------------------|
| `about.jpeg` in **`src/assets/`** (not this folder) | About portrait | 4:5 portrait |
| `beyond.jpg` | Beyond Tech section | 4:5 portrait |
| `project-samvadd.jpg` | Work → Samvadd.AI | 16:10 landscape |
| `project-investizen.jpg` | Work → Investizen | 16:10 landscape |
| `sprint-sih.jpg` | Sprints → Smart India Hackathon | 4:3 |
| `sprint-hackathon-1.jpg` | Sprints → 24h hackathon (large) | 4:3 |
| `sprint-hackathon-2.jpg` | Sprints → gallery (tall) | 3:4 |
| `sprint-hackathon-3.jpg` | Sprints → gallery (tall) | 3:4 |

You can use `.png` or `.webp` instead — if so, update paths in `src/content.ts`.

After adding files, run `npm run dev` and refresh the browser. Images are served publicly at `/images/...` and included when you deploy.
